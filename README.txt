Если вам понадобится вносить изменения в иходный код FEW, Пожалуйста, публикуйте их!

Пожалуйста, делайте это следующим образом:
1. зарегистрируйтесь на GutHub
2. создайте форк проекта https://github.com/sargash/few
3. клонируйте форк на свою рабочую машину
4. настройте синхронизацию с Вашей(корпоративной) системой контроля версий
5. вносите изменения и публикуйте их на GitHub
6. присылайте pull-request'ы в основной репозиторий с исправлениями/улучшениями

Такой подход позволит вам гибко интегрировать новые исправления из основного репозитория FEW, 
а так же, гибко обмениваться исправлениями между разными разработчиками и разными ветками FEW.

Вместе, мы сможем улучшать проект быстее!
